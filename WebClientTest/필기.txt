Markup Language
- SGML
	- Standard Generalized Markup Language
	- 모든 ML의 시조
	- 메타 언어 : 또 다른 언어를 정의하는 기능을 가진 언어
- HTML
	- SGML 파생 언어
	- 표현하는 기능
	- 무언가를 출력하는 역활
	- HTML1.0 ~ HTML2.0 ~ HTML4.01(***) ~ HTML5(2014)
- XML
	- eXtensiable Markup Language
	- 확장 가능한 마크업 언어
	- SGML 파생 언어
	- 메타 언어 > 데이터를 저장하는 용도의 언어(주)
- XHTHML
	- XML _ HTML
	- eXtensible Hyper Text Markup Language
	- XHTML 1.0 ~ XHTML 2.0(X)
